# Using GenAI to review rules and structure

## Preparation

1. I made a game. Can you suggest any gaps?
2. Is there anything unclear about the play format?
3. I have created a variant with normal playing cards and coins? Can you see if it is understandable?

## Simulation
4. Can you simulate a game using playing cards and the flow of play?
5. How much time would one round for 4 people take?

## New Variant
6. I have created a new version using regular playing cards. Can you review the rules?
